 Salesforce Platform

Salesforce is not only a CRM tool. It also provides a platform where businesses can build custom applications based on their requirements.

*The platform includes tools for:
1. Sales management
2. Customer support
3. Automation
4. Reporting
5. Mobile applications
6. AI features


 Important Terms

1. App
An app is a collection of tabs, objects, and features used for a specific business process.

2.Object
Objects are similar to database tables used to store information.

3. Record
A record is a single entry inside an object.

4. Field
Fields are used to store specific information in records.

5. Org
Org means organization. It is the Salesforce environment used by a company.

No-Code Development:

No-code development allows users to build applications using drag-and-drop tools without writing code.

Programmatic Development:

Programmatic development uses coding languages like Apex and Lightning Web Components to build advanced features.
